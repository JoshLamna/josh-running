One-click sync (Tampermonkey)
Adds a permanent ⚡ Sync Training Plan button to Garmin Connect. Auto-detects your auth token — no DevTools needed.
  1. Install Tampermonkey from Chrome Web Store (free — you said it's already installed ✓)
  2. Download the userscript below
  3. Drag the downloaded .user.js file onto the Tampermonkey dashboard and confirm install
  4. Visit connect.garmin.com — click the orange button that appears
DevTools script (desktop fallback)
Copy the script, paste into the DevTools Console on connect.garmin.com. Reflects any session reschedules made in this app.
